Project Overview

Website for Cimo :
Cimo is a fake clothing brand with medium-sized office based in Indonesia. The company could ship their products to Singapore and Malaysia. 

Objective :
Creating a website for such company that is also a platform or tools for the employees to create/manage content within the website.

Roadmap :
- Authorization (Register / Login)
- Account Verification
- Display Products
- Product Sections (New Arrivals, Men, Women)
- Search and Filtering
- Pagination
- Responsive
- Display Currency based on Selected Region
- Customer Features :
  - My Account (Customer Details)
  - Add to cart
  - Uploading Proof of Payment
  - Checkout / Transactions
- Admin Features :
  - Manage Products
  - Analytics
  - Completing Transactions
  - Emailing receipts
